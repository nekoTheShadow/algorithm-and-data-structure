First Fit法によりX本のビンに詰めることができたとする。
i番目のビンに入っている品物の合計をw[i]とすると、w[i]+w[i+1] > 1 が成立する。
  a[0] + a[1] + ... + a[n-1] 
    = w[0] + w[1] + w[2] + w[3] + ... + w[x-2] + w[x-1]
    = (w[0]+w[1]) + (w[2]+w[3]) + ... + (w[x-2]+w[x-1])
    > X/2 (*1)

最適なアルゴリズムによりY本のビンに詰めることができたとする。
このとき以下が成立する。
  (a[0] + a[1] + ... + a[n-1]) / Y <= 1
    <=> a[0] + a[1] + ... + a[n-1] <= Y (*2)

*1と*2をまとめると X/2 < a[0] + a[1] + ... + a[n-1] <= Y
よって X/2 < Y となり、First Fit法は2-近似アルゴリズムであるといえる。
